google Map api 연동 google Map api는 위치 기반 지도를 서비스하기 위한 지도 기반 앱을 만듬 geolocation api 속성 center : 위도와 경도 설정 zoom : 지도에서 표시의 크기 나타냄 설정범위는 0~22 mapTypeId : 지도 유형 google.maps.MapTypeId.ROADMAP : 도로 지도뷰를 표시 (기본 지도 유형) google.maps.MapTypeId.SATELLITE : 위성 뷰를 섞어서 표시 google.maps.MapTypeId.HYBRID : 일반 뷰와 위성 뷰를 섞어서표시 google.maps.MapTypeId.TERRAIN : 지도 정보를 기반으로 실제 지도를 표시 google.maps.LatLng : 위도와 경도를 나타냄 (latitude longitude) google.maps.Map : 지도를 나타낸다 google.maps.Marker : 마커를 표시한다 setMap()메서ㄷ드를 호출해 마커를 추가함 지오로케이션 api 지오오케이션은 자바스크립트 api 를 이용해서 사용자의 위치 정보를 제공하는 api이다 1.위치를 수집하는 방식 wifi 네트워크와 ip주소 gps를 이용해 현재의 위치 정보 받음 2.개인 정보 볼호 사생활 침해 등의 이유로 사용자의 동의 받아야함 navigator.geolocation.getCurrentPosition(success, [error],[option]); 메서드는 기기가 있는 곳의 현제 위치를 받아옴
