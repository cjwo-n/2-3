Promise: Promise 객체는 비동기적으로 실행될 코드 블록을 나타냅니다. Promise 객체는 세 가지 상태를 가질 수 있습니다: 대기 중(pending), 이행됨(fulfilled), 거부됨(rejected). 대기 중인 Promise 객체는 비동기 작업이 완료될 때까지 기다리며, 이행됨 상태나 거부됨 상태가 되면 그에 따른 콜백 함수가 실행됩니다. resolve: Promise.resolve()는 fulfilled 상태인 Promise 객체를 반환합니다. Promise.resolve()는 주어진 값이 Promise 객체인 경우 그 값을 그대로 반환하고, 그렇지 않은 경우 주어진 값을 이용하여 fulfilled 상태인 Promise 객체를 생성합니다. reject: Promise.reject()는 rejected 상태인 Promise 객체를 반환합니다. 주어진 값을 이용하여 rejected 상태인 Promise 객체를 생성합니다. then: Promise.then()은 Promise 객체가 이행될 때 실행될 콜백 함수를 등록합니다. then() 메서드는 Promise 객체를 반환하며, 이행될 때 콜백 함수의 반환값이 다음 then() 메서드의 인자로 전달됩니다. catch: Promise.catch()는 Promise 객체가 거부될 때 실행될 콜백 함수를 등록합니다. catch() 메서드는 Promise 객체를 반환하며, 거부될 때 콜백 함수의 반환값이 다음 then() 메서드의 인자로 전달됩니다. finally: Promise.finally()는 Promise 객체가 이행되거나 거부될 때 실행될 콜백 함수를 등록합니다. finally() 메서드는 Promise 객체를 반환하며, 이행될 때나 거부될 때 모두 실행됩니다.
